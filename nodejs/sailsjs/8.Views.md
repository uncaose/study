## Creating a view
- Default EJS([Embedded Javascript](http://ejs.co/)) Engine
- `views/viewfile.js`

## Compiling a view
- router.js 

```javascript
{
  'get /': {
    view: 'pages/homepage'
  },
  'get /signup': {
    view: 'pages/signup/basic-info'
  },
  'get /signup/password': {
    view: 'pages/signup/choose-password'
  },
  // and so on.
}
```
